Periods 时段
定义时间窗口及其行为: 采集(collect)、分析(analyze)、推送(push)、报告模式(report_mode)、AI 模式(ai_mode)。支持跨午夜时段。
Day Plans 日程
将多个时段组合为一日计划，例如 "workday"(工作日)和 "weekend"(周末)可有不同的采集和推送策略。
Week Map 周映射
将日程分配到星期 (1=周一…7=周日)。例如: 周一至五用 workday 计划，周六日用 weekend 计划。

DEPLOYMENT
部署模式
💻
本地 Python
使用 uv 包管理器直接运行。适合开发调试和个人使用。支持单次执行和手动触发。
uv run trendradar
📦
Docker Compose
两个服务: trendradar (采集+推送) + trendradar-mcp (MCP 服务)。supercronic 定时调度，支持 cron/once 模式。
docker compose up -d
⚡
GitHub Actions
CI/CD 自动化。使用 GitHub Secrets 管理敏感配置，按 cron schedule 触发工作流。
workflow_dispatch / cron
